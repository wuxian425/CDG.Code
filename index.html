介绍


COVID-19 Data Getter 是一个简单的，具扩展性的项目，整体使用 Turbowarp 进行开发，并使用 Turbowarp Packager 进行压缩打包成网页文件。

COVID-19 Data Getter 使用 MIT License 进行开源，您可以使用 Turbowarp + Fetch扩展 进行编辑并打包成网页文件。但是，请遵守 MIT License 规范 ，否则我方有权停止开源。
